Secure. Optimized. Ready.

PsychopompOS is a Debian 13 (Trixie) based Linux distribution designed for server environments and cybersecurity operations. It comes preloaded with a hardened configuration, essential server software, and powerful administration tools to get you up and running quickly and securely.

Key Features

Debian 13 Base

Built on the rock-solid foundation of Debian 13 with full repository compatibility and long-term support.

Preinstalled LAMP Stack

Apache2, MariaDB, PHP with common modules ready to deploy your web applications instantly.

Secure Mail Server

Postfix, Dovecot, and Roundcube preconfigured for secure email hosting.

Security Tools

UFW firewall, Fail2ban, nmap, tcpdump, arp-scan, mtr, and more for network defense and monitoring.

Development Ready

Python 3, OpenJDK 17, Node.js, npm, Git, and build-essential for software development and CI/CD.

Offline Admin Tools

phpMyAdmin, Webmin, and tgpt CLI tool bundled as local `.deb` packages.
